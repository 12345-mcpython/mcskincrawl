# mcskincrawl
mc皮肤解析下载
Minecraft skin parse and download
# Usage:
`java -jar mcskincrawl-0.0.1.jar <your minecraft user name>`

# Or:
```
   import com.mc.McSkinCrawl;

   public class Skin {
        public static void main(String[] args) {
    	      McSkinCrawl.gen("muyu_");
        }
   }
```
